# Обзор проекта

**CodeSnip** — веб-приложение на **Laravel 12** и **Livewire** для хранения, редактирования, поиска, версионирования и обмена фрагментами кода (сниппетами).

## Целевая аудитория

Разработчики, которым нужен личный или командный каталог сниппетов с нормальным редактором, историей изменений и удобным поиском.

## Ключевые возможности

- **Сниппеты**: создание, правка, удаление; язык; теги; публичная / приватная видимость; стабильный UUID и публичная страница.
- **Редактор**: CodeMirror 6, горячие клавиши (в т.ч. сохранение), черновики в `localStorage`, опциональный автосохранение, форматирование JSON, подсветка проблем (JSON, хвостовые пробелы).
- **Ревизии**: снимки при значимых изменениях, список версий, сравнение (с текущей / с предыдущей), откат одной кнопкой.
- **Поиск**: фильтры по тексту, языку, тегам, видимости, датам; сортировка; **сохранённые поиски** на пользователя.
- **Организация**: папки, связь many-to-many со сниппетами; **smart-коллекции** с правилами в JSON.
- **Шаблоны сниппетов**: заготовки с подстановкой переменных `[[имя]]`.
- **Импорт / экспорт**: архив ZIP, JSON, обмен через **GitHub Gist** (личный токен пользователя).
- **ИИ (локально)**: интеграция с **Ollama** — подсказки, саммари, объяснение кода, тесты, фоновая генерация тегов; выбор модели в профиле.
- **Интерфейс**: Livewire + **Flux UI**, тёмная/светлая тема, локали **en** / **ru**.

## Технологический стек

| Компонент | Версия / примечание |
|-----------|---------------------|
| PHP | ^8.4 |
| Laravel | ^12 |
| Livewire, Volt | полноценные страницы настроек через Volt |
| Flux | UI-компоненты |
| Vite, Tailwind | фронтенд-сборка и стили |
| База | SQLite по умолчанию в `.env.example`; в Docker — MariaDB |
| Очереди | `database` или `redis`; пакет **Horizon** для Redis |
| Поиск | **Laravel Scout**; драйвер по умолчанию `collection`, опционально Meilisearch |
| Очередь задач | `GenerateTagsJob` и др. |
